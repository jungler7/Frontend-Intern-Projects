# Frontend Intern Projects

This repository contains beginner-level frontend projects developed using
HTML, CSS, and JavaScript

The focus of these projects is on clean structure, simple logic, and
readable code.
<a href="portfolio/index.html">Portfolio Website</a>
<a href="todo/index.html">To-Do List App</a>

---

## Projects Included

### 1. Portfolio Website
A simple static portfolio website that introduces basic personal
information, skills, and projects.

**Features:**
- Semantic HTML structure
- Clean and simple CSS styling
- Multiple sections (About, Skills, Projects, Contact)

---

### 2. Form Validation Project
A basic login form with client-side validation using JavaScript.

**Features:**
- Email and password input fields
- Validation using JavaScript
- Error messages displayed for invalid inputs

---

### 3. To-Do List App
A simple to-do list application that allows users to add and mark tasks
as completed.

**Features:**
- Add new tasks
- Mark tasks as completed
- JavaScript DOM manipulation

---

## Technologies Used
- HTML5
- CSS3
- JavaScript (Basic)
